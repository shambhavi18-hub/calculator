# 🧮 Python Calculator

A menu-driven command-line calculator built with Python. This project was developed as part of my Python learning journey to strengthen my understanding of functions, loops, exception handling, and user interaction.


## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ➗ Modulus
- ⬆️ Power
- // floor
- 📋 Menu-driven interface
- ⚠️ Handles invalid number inputs
- 🚫 Prevents division by zero
- 🔄 Continue/Exit functionality
- 💻 User-friendly terminal interface

- Git
- GitHub



## 📚 Concepts Practiced

- Functions

- User Input (`input()`)
